# FPGA Temperature Controller
Introductory academic project focused on understanding basic digital design concepts using VHDL and FPGA technology.

# Description
This project implements a basic temperature controller using VHDL, designed to run on an FPGA development board. The main goal of the project was to gain hands-on experience with hardware description languages and understand how digital logic designs are implemented and deployed on FPGA hardware.

# Project Focus
- Learning the basics of VHDL syntax and structure
- Mapping a digital design onto FPGA hardware
- Basic simulation and hardware implementation workflow

# Technical Details
- Written in VHDL
- Simple control logic based on temperature thresholds
- Designed for educational purposes
- Introductory-level FPGA project

# Technologies Used
- VHDL
- FPGA
